-----------------------------------------------------------
Pre-requisite for using Amazon Locker Recommendation Engine
-----------------------------------------------------------

1. sudo apt install python3-pip

Machine learning

2. pip3 install pandas
3. pip3 install -U scikit-learn
4. python3 -m pip install -U matplotlib
5. sudo pip3 install seaborn

GUI - Django

1. pip3 install virtualenv
2. sudo apt install virtualenv
3. mkdir django-apps
4. cd django-apps 
5. virtualenv env
6. . env/bin/activate
7. pip3 install django
8. django-admin --version


OR

sudo apt-get install python3-django
